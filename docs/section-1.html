Teaching with the narrative and model in NASEM (2021) ‘Nutrient Requirements of Dairy Cattle’

Chapter 6 3-2

$$ \[\begin{align} \text{GE_DM of feed} &= (0.042 * \text{NDF_DM}) + \\ & \ \ \ \ \ (0.0423 * \text{Starch_DM}) + \\ & \ \ \ \ \ (0.040 * \text{ROM_DM} )+ \\ & \ \ \ \ \ (0.094 * \text{FA_DM}) + \\ & \ \ \ \ \ 0.0565 *( \text{CP_DM} - \text{sNPNCPE_DM}) + \\ & \ \ \ \ \ (0.0089 * \text{sNPNCPE_DM}) \\ \end{align}\] $$

Where feed fractions are expressed as a percentage of DM.

Try it for yourself

NDF_DM = 0.4
Starch_DM = 9
ROM_DM = 9
FA_DM = 9
CP_DM = 9
sNPNCPE_DM = 8

# Can you write a function that will take these inputs
# in R. 

GE_DM <- function(NDF_DM, Starch_DM, ROM_DM, FA_DM, CP_DM, sNPNCPE_DM) {
  GE_DM = 0.042*NDF_DM +0.0423*Starch_DM + 0.040*ROM_DM + 0.094*FA_DM + 
    0.0565 * (CP_DM - sNPNCPE_DM) + 0.0089*sNPNCPE_DM
  return(GE_DM)
}
